# CyberFest 2026 - Digital Defence Club

Welcome to CyberFest 2026, the premier cybersecurity event organized by the Digital Defence Club. This website showcases our upcoming Hackathon and CTF competitions.

## Features

- Modern, responsive design with cybersecurity theme
- Dedicated sections for Hackathon and CTF events
- Smooth navigation between sections
- Mobile-friendly layout
- Interactive elements and animations
- Digital Defence Club branding throughout

## Structure

- `index.html` - Main website structure
- `styles.css` - Styling with theme colors from Digital Defence Club
- `script.js` - Interactive functionality

## Design Elements

The website incorporates the Digital Defence Club branding with:
- Purple (#8b3dff) and teal (#00c4cc) theme colors
- Cybersecurity-inspired visual elements
- The club's logo throughout the site
- Dark theme for a modern, tech-focused look

## Events

### Hackathon
- Real-world cybersecurity challenges
- Expert mentors and guidance
- Networking opportunities
- Prizes worth ₹5,00,000

### CTF (Capture The Flag)
- Multiple challenge categories
- Jeopardy-style competition
- Various difficulty levels
- Prizes worth ₹2,50,000

## How to Use

Simply open `index.html` in any modern web browser to view the website. The site is fully responsive and will work on desktop, tablet, and mobile devices.
